# Bank Account Management System

## Description
The Bank Account Management System is a Python-based application that allows users to create accounts, deposit and withdraw money, check account balances, and view transaction history. Account information is stored using CSV files.

## Features
- Create Account
- Display Accounts
- Search Account
- Deposit Money
- Withdraw Money
- Check Balance
- View Transaction History
- Save Account Records to CSV

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Inheritance
- Abstraction
- Exception Handling
- File Handling

## Files Included
- bank_management.py
- Bank_Management_System.ipynb
- accounts.csv

## Technologies Used
- Python 3
- Jupyter Notebook
- CSV File Handling
